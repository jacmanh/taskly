# Taskly Design System

Design system moderne avec Tailwind CSS pour l'application Taskly.

## Palette de couleurs

### Primary (Bleu fade)
Couleur principale de l'application - un bleu fade moderne et élégant.

- `primary-50` à `primary-950` : Nuances du bleu primaire

### Secondary (Gris neutre)
Couleurs secondaires pour les textes et arrière-plans.

- `secondary-50` à `secondary-950` : Nuances de gris

### Accent (Violet)
Couleur d'accentuation pour les éléments spéciaux.

- `accent-50` à `accent-950` : Nuances de violet

### Success (Vert)
Pour les messages de succès et les actions positives.

- `success-50` à `success-950` : Nuances de vert

### Warning (Orange)
Pour les avertissements.

- `warning-50` à `warning-950` : Nuances d'orange

### Error (Rouge)
Pour les erreurs et les actions destructives.

- `error-50` à `error-950` : Nuances de rouge

## Composants CSS

### Boutons

- `.btn` : Classe de base pour tous les boutons
- `.btn-primary` : Bouton principal avec le bleu fade
- `.btn-secondary` : Bouton secondaire
- `.btn-outline` : Bouton avec bordure
- `.btn-ghost` : Bouton transparent

### Cartes

- `.card` : Carte avec ombre douce et bordures arrondies

### Formulaires

- `.input` : Champ de saisie stylisé
- `.label` : Label pour les formulaires

## Utilisation

Les classes Tailwind sont disponibles directement dans les composants React. Utilisez les classes de couleur comme `bg-primary-500`, `text-secondary-900`, etc.
